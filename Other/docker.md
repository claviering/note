# docker

## 重启容器

`docker restart [CONTAINER ID]`

## 停止容器

`docker stop [CONTAINER ID]`

## 列出容器

正在运行的

`sudo docker ps`

全部容器

`sudo docker ps -a`

## 删除 image

`docker image rm [IMAGE ID]`

## 删除 container

`docker container rm [CONTAINER ID]`

## 列出 container

`docker container ls -a`

## 列出下载的镜像

`docker image ls`

## Execute Docker image

`docker run hello-world`

## 登陆

`docker login www.example.com`
