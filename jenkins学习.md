# jenkins 学习

一键部署到服务器

## 安装

安装 java 8

## jenkins 下载

mac OS 下载目录 Application/Jenkins/jenkins.war

复制 jenkins.war 去别的目录

## 运行

java -jar jenkins.war --httpPort=9090

## 配置

浏览器打开 localhost:9090 

复制密码 /Users/linweiye/.jenkins/secrets/initialAdminPassword
